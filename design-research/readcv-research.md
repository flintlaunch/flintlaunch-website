# read.cv — Design Research

> Note: Site returned 402 (deployment paused) during scrape. Analysis based on known design patterns from prior observation.

## Color Usage
- Background: White or very near-white
- Accent: `#79FFE1` — mint/teal (matching scraped data)
- Text: `#000000` — true black
- Contrast-first: black on white, no in-between

## Typography
- System fonts (SF Pro) — intentional anti-decoration
- Body: 20px — large, readable, editorial pace
- Minimal hierarchy: large body text IS the hierarchy
- No display font tricks — typography through weight/size alone

## Layout Patterns
- Zero border radius on many elements (sharp, editorial)
- Dense whitespace — "empty" feeling that feels curated
- Profile cards: minimal info, maximum breathing room
- Typography-first: copy leads, visuals follow

## Animation Style
- Near-zero animations — confidence without motion
- Hover: immediate color change, no transforms
- Page transitions: fade only

## Hover Effects
- Text links: underline appears
- Cards: border color change only
- Nothing moves — very deliberate

## Scroll Interactions
- Linear scroll — no parallax, no reveal animations
- Content is there waiting, not performing

## Design Philosophy
> Restraint as luxury. The less that moves, the more each word matters.
